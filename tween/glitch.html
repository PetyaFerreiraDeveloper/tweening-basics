A common animation deffect

In the example below, if we use only the from method from gsap, if we start hovering very fast the inner circle will get smaller and smaller, because it will have a smaller refference point to calculate the scale from.

To fix this, we can use the fromTo method, and set the scale from 0 to 1 always, so the inner circle will always have the same refference point to calculate the scale from.
